Check out my projects below.

StockSurfer

StockSurfer is my latest project I've been working on a developer. This is a full stack web application using React and Plotly.js to create an app that tracks data for over 2400 stocks in the New York Stock Exchange. The app also features a built-in payment form using Stripe for React to accept a user payment for the Pro version of the app. The pro version includes additional features.

Snuggle Reads Book Subscription

SnuggleReads is a user-friendly and comprehensive website designed for book enthusiasts who want to keep a record of their reading journey. This platform offers a book catalog that enables users to organize, track, and explore their literary adventures effortlessly.

Movies.KP

Movies.KP is a comprehensive movie application that allows users to search up any movie amongst the a database of over 280,000 movies using the OMDB API. Users can then click on the selected movies to view more information about them, making this this perfect movie info site for all your movie fun facts!

Canine Club

A full-stack web application created with the goal for users register their information and assign them with their own unique avatar of a dog! This was an exercise in fetching API data and passing props.

To-Do List

This is a simple and attractive to do list that does exactly what it says. You can add as many tasks as you wish, mark them as done, and delete them when you're finished. You can also clear out entire lists without having to delete each task individually. This individual project has a ton of room to be expanded further!

H&F

This is my first project in my journey as a developer. It is a front-end application that helps steer users in the right path on improving their health and fitness. H&F incorporates 2 APIs, one for fetching exercises based on user input, and an API to fetch recipes based on user input. This allows a user to kickstart their fitness goals by giving relavant suggestions.
